# 결제 모듈 리뷰

## 고객사 요구사항

> 이메일에 작성된 요구 사항 외에 피그마 시안에 있는 세부적인 요구사항을 추가해주시면 됩니다.

1. 여러 개의 카드를 등록할 수 있게 해주세요.
2. 카드의 민감한 정보가 바로 노출되지 않도록 해주세요
3. 이 결제 모듈의 경우, 상품 목록에서 담기 버튼 바로 옆에 '결제' 버튼을 추가하여 동작할 수 있게 만들어주세요.

## 요구사항에 따른 기능 동작 확인

### 1. 카드 등록

- 여러 개의 카드를 등록할 수 있습니다.

### 2. 민감 데이터 마스킹

- 카드 목록에서 민감 정보가 마스킹 처리됩니다.

### 3. '결제' 버튼

- 상품 카드에 '결제' 버튼을 추가했습니다.
- 버튼 클릭 시, 카드 목록 페이지로 이동합니다.
