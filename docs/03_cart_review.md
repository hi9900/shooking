# 장바구니 페이지 리뷰

## 고객사 요구사항
1. 금액을 정확하게 계산해서 보여주세요
2. 총 금액이 10만원 미만이라면 배송비 3,000원, 10만원 이상은 배송비 무료입니다.
3. 뒤로가기 버튼을 이용해 메인 상품 목록 페이지로 이동할 수 있게 해주세요.
4. 결제하기 버튼을 누르면, 결제 모듈을 모달로 사용할 수 있게 연동해주세요.

## 요구사항에 따른 기능 동작 확인

1. 금액 계산

- '총 금액'(total)은 '상품 금액'과 '배송비'의 합계로 정확히 계산됩니다.

2. 배송비 정책

- '상품 금액' 100,000원 미만 시 "3,000원", 100,000원 이상 시 "무료" (0원)가 표시됩니다.

3. 페이지 이동 및 결제

- '뒤로가기' 버튼 클릭 시, 이전 페이지(상품 목록)로 이동합니다.

4. 결제하기

- '결제 모듈'로 이동하는 '결제하기' 버튼이 있습니다.
